## (e) Authoring Batch Plan — Date-Led, Then Scale

### Quality bar
- **Warm Premium voice** — sensory, assured, atmospheric (never hype or pity)
- **Technique-forward** — teach the skill, not just the steps
- **Scalable** — ingredients written per-serving-friendly (e.g. "1 steak, ~250g")
- **Accurate tags** — showstopper, date-night, one-pan, 30-min, high-protein
- **Gear check** — one pan, sheet tray, one pot, tongs (no specialist equipment)

### Batch 1: Date-led pilot (8 recipes) — TEST THE SYSTEM
**Goal:** Prove the system works. Write 8 date-night library dishes using `prompts/write-recipe.md`. Verify frontmatter parses, voice lands, tags filter correctly.

**Priority order (impact + technique gap-fill):**
1. Bouillabaisse (French) — seafood theatre, multi-stage
2. Bibimbap (Korean) — bowl assembly, gochujang technique
3. Pho Bo (Vietnamese) — stock-making, noodle soup
4. Moussaka (Greek) — layering, béchamel
5. Osso Buco (Italian) — braise, gremolata
6. Korean BBQ Beef (Korean) — marinade, lettuce wraps
7. Banh Mi (Vietnamese) — fusion, pickling
8. Doro Wat (Ethiopian) — berbere spice, injay

### Batch 2: Solo scaling (20 recipes) — BUILD THE ENGINE
**Goal:** Flesh out the solo engine. Focus on weeknight efficiency, skill-building, global comfort.

**Cuisine spread:** Italian (4), Japanese (3), Thai (3), Indian (3), American (2), French (2), Mexican (2), Chinese (1).

### Batch 3: Host centerpiece (10 recipes) — GENEROSITY MODE
**Goal:** Prove the library scales. Write the interactive, communal dishes.

**Priority:** Peking Duck, Paella, Moroccan Tagine, Pozole, Dim Sum, Mole Poblano, Nigerian Jollof Rice, Tamales, Jamon Iberico, Potbrood.

### Batch 4: Remainder (50 recipes) — COMPLETE THE 100
**Goal:** Finish the library. Mix of solo/date/host to hit target ratios.

**Order:** Fill remaining gaps, maintain 50/35/15 balance, ensure full pan-global coverage (minimum 3 dishes per major cuisine cluster).

### Post-batch verification
After each batch: verify frontmatter parses, occasion filters work, serving-scaling UI recalculates correctly, voice remains Warm Premium.
